SOFTWARE HOUSE Application

Home page

Main page of the application. Here are clock and date or localization

This page shows up right after logging in. There are main sections of the aplication. Home page includes user's greetings, clock, current date, weather and main calendar.

Reminder

Simple reminder. Added task are saving in package JSON

The reminder includes adding tasks. Right after writting date, name of the task and discription user should click "Add" button and added task will be appear on the right side. Every task is saving in package JSON of the browser. That means that if user turn the application off, tasks will be saved in this browser in package JSON.

Projects deadlines

Adding project deadline and showing all of them in the calendar

The projects deadline section contains the calendar of added projects. User is saving a project deadline date in the calendar and right after that project deadline section will include this date.

Completed projects

Adding completed projects to the database

The completed projects section includes the database of added completed projects. User can add completed project. After that user can extend, delete or edit new record. Completed projects section includes search bar to automate using the app.

Company invoices

Adding company invoices system combined with database

The company invoices section contains the database of the added invoices to the system. Every record is saved. After adding a new invoice, user can download it from the app.

Employees database

Adding Employees with image to the system

The employees database contains a system of added employees into the app. During adding a new employee, user should add his picture, name, specialty, and e-mail. Every added record is saved in the database. User can extend, delete or edit every record of the employee in the application.
